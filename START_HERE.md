# Start Here

## What this version does

Upload 1–10 annual reports. They may belong to one company or several companies. The app detects each company, groups matching reports and generates one three-statement Excel workbook per company.

## Deployment order

1. Upload every file in this project folder to the root of your GitHub repository.
2. Deploy the repository on Render using `render.yaml`.
3. Confirm the Render backend URL in `docs/config.js` (the current URL is already filled in).
4. Enable GitHub Pages from `main` → `/docs`.
5. Open the GitHub Pages link and click **Wake the Numbers**.
6. Test with one native-text annual report first, then test a mixed-company batch.
